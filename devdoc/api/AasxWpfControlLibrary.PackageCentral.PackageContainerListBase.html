    Class PackageContainerListBase

    This class takes over source code from the legacy AasxFileRepoBase. It therefore realizes a list of items, which associated some ids (AasId, AssetId, SubmodelIds) with AASX Package files. In DEC 2020, the PackageComtainers were introduced to leverage the hosting of AASX Package files and to enhance and abstract them by providing online connections and event management. In JAN 2021, the idea is now to join these capabilities. This class implements a list of PackageContainerRepoItem, so it can provide information, which AASX Package file is to be loaded and hosted and functinality, HOW this can be done. This class is intended to be a base class, so classes for local repos, AAS repos, AAS registries are deriving from it.
