    Class FormDescSubmodel

    An specialization of FormDescListOfElement. Defines a total Submodel to be represented by the form, that is, no outer structures tha form instances will be in the Submodel. The Plugin's will match to the SemanticId of the Submodel, therefore it has to be present.
