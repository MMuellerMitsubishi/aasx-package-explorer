    Class PackageCentral

    This class (more: a singleton) centralized the information on open package environments for the Pakcage Explorer. It hold Main and Aux packages, but also the open file repositories. In future, it might store multiple open packages for the same time, as AASX Server might require, and Main might only an index into these.
