    Class AasEventMsgEnvelope

    Outer class for any AAS event. AAS events shall interoperably exchage asynchronous information between different execution environments "operating" AASes. This basic event class is described in AASiD Part 1.

    Note: This envelope is able to carry one or multiple even payloads.
