    DefinitionsExperimental

    This class holds definitions, which are preliminary, experimental, partial, not stabilized. Aim is to provide a single source of definitions, which can be used by implementations such as plugins, to develop some functionalities. Later, these definitions shall be replaced by substantial definitions, e.g. imported from JSON.
